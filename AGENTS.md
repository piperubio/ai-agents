This is a project for managing IA agents configurations and templates.

## Skills

- For create or validate skills always use the `skill-creator` tool.
- All skills are stored in the `skills/` folder.
- Each skill must have a name with lowercase and hyphens, for example: `consulting-project-planner`.
- Each skill muut have a description with what it does and when to use it. Should include specific keyworkds that help agents identify relevantt tasks.
