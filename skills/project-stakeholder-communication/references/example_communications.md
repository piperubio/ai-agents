# Example Communications

Use these examples to validate the Stakeholder Communication skill.

## Example 1 — Executive decision request (high urgency)

Input
```yaml
communication_context:
  audience: executive
  purpose: decision_request
  urgency: high
```

Expected highlights
- Tone: executive
- Clear objective, risks, options, decision requested, next steps

## Example 2 — Internal status update (low urgency)

Input
```yaml
communication_context:
  audience: internal
  purpose: status_update
  urgency: low
```

Expected highlights
- Tone: neutral/collaborative
- Summary of progress, minor issues, immediate next steps
