# IA Agents Configurations and Templates

This repository centralizes the configurations, templates and examples that I use in my projects. It aims is hava reutilizables parts for principal agents or modes, subagents, skills and mcp servers. 


## Structure

- `agents/` - configurations and templates por principal agents or modes
- `subagents/` - specialized agents with specifict goals and instructions
- `skills`- agent skills with dynamic load for specific tasks.
